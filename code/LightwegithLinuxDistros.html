Linux is an open-source operating system family that has a huge variety of operating systems (nicknamed distributions or distros) that is tailored to specific users. In this blog post, I will be covering ultra-lightweight Linux distros, specifically operating systems that run under 200 MB RAM. If you have a computer that once ran an old windows/apple OS version, you may be interested in this article.

1. Trisquel Mini

Trisquel Mini is a part of a group of Trisqueli distributions, which is focused on both home use and being lightweight. The boot process is pretty quick too. Upon successful boot of the desktop environment (LXDE), you can see that in the image below, Trisquel Mini packs a punch on being light on system resources.

Trisquel Mini's interface looks simplistic aswell. Though, it may be too basic for some.

 

SUMMARY:

If you support FOSS software and actively try to avoid proprietary software, you may like Trisquel Mini as it believes in FOSS software and only uses FOSS software.

Trisquel Mini uses LXDE, an outdated desktop environment abandoned by it's owner, Lubuntu.

Trisquel Mini has a few pre-installed applications, which promotes a clean desktop.

Triquel Mini is based on Ubuntu/Debian repositories, meaning that all of your favorite goodies from there can be installed on Trisqueli Mini as well.

Trisquel's interface may be too basic for some people.

Trisquel's reviews on Distrowatch can be a little mixed. The top complaint by far is the difficulty to connect to WiFi.

2. AntiX Linux

AntiX Linux is another ultra-lightweight distribution that is filled with programs. However, these programs are actually useful, eg. a live usb maker for distro hoppers and an encoder for if you want to modify images or videos.

Anyways, enough of rambling about the software on AntiX, lets check out the RAM usage on AntiX 64-bit. Since it has a variety of desktop environments, I will try the minimal iceWM manager.

 

SUMMARY:

AntiX is very user-friendly, coming with full self-support documentation and a load of programs

The abundance of programs may put a strain on system storage.

AntiX has a LOT of (lightweight) desktop environments, iceWM, JWM, etc., theres a bunch of them to choose!

Based on Ubuntu/Debian repositories.

Great-looking UI.

A user on AntiX's sourceforge page complains that it can be very troublesome to connect to the internet if connecting via WiFi.

3. Puppy Linux

Puppy Linux is a veteran in serving the super-old-PC distro world. Living for 14-15 years, this distro beats Lubuntu's age and still provides hasty support and documentation. So far, the 32-bit versions can go almost around 50mb using Bionic Puppy while 64-bit goes around 80MB-100MB. Puppy Linux has a few programs, notably for the core functions for everyday computing, such as a media player/viewer, a calendar, email client, office software, etc.

For the examples below, I will be using Bionic Puppy, which is by far the most lightweight distribution. Other ones, like Fossa Pup, can take up a little bit more RAM.

5. Certain Desktop environments of Porteus

Porteus is a linux distribution focused on portability. Straight off the bat, this means to us that Porteus will run live and will be focused on running live. Porteus currently has 7 desktop environments for install, with the lightweight DE's being Openbox, LXQt, and LXDE. We ran the LXQt version on 64-bit as we tested Porteus.

Not bad.

SUMMARY:

If you have a strain on storage but can use things such as a USB drive/CD/DVD, you may like Porteus as it can run directly from it.

Interface can feel a little dated on some DE's.

Based on Slackware, so all your Slackware stuff can work on Porteus.

A user on Slant.co complains that Porteus's package manager breaks due to a broken repository.
